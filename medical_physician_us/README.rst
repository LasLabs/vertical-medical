==========================================
Medical Physician - US Locale - DEPRECATED
==========================================

This module has been deprecated in favor of ``medical_practitioner_us`` and
should only be used to update a prior ``medical_physician_us`` install. To
avoid issues when updating, please follow the instructions in the
``medical_physician`` README.

Bug Tracker
===========

Bugs are tracked on
`GitHub Issues <https://github.com/OCA/vertical-medical/issues>`_. In case of
trouble, please check there if your issue has already been reported. If you
spotted it first, help us smash it by providing detailed and welcomed feedback.
