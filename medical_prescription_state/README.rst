==========================
Medical Prescription State
==========================

This module extends ``medical_prescription`` by adding support for Kanban
stages to prescription orders and order lines.

Usage
=====

#. Go to ``Medical > Medicine > Prescription Orders`` or
   ``Medical > Medicine > Prescription Lines``
#. Notice that the records are displayed in a Kanban view and that some stages
   have already been defined
#. Add/remove stages and move records between them as needed
